+ Where to start!

Welcome!! I created these apps to make my life easier when posting graphics and images in my LifeJournal. And to easily personalized profiles.

I'm sharing them here as they might be useful for others too.

-- Icontable Generator --

You can create personalized tables, not only for icons but for any bigger graphics too, creating thumnails linking to the graphics.

-- Picspam Generator --

Create tables to post pictures or any kind of graphics. And add text in between the pictures

-- Profile Generator --

Pick an existing layout from the existing list to generate a profile

-- Votes Counter --

Use to count votes when running Icontest Comms or any kind of Challenge Comm where you use polls to decide the winners.
